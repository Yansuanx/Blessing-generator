## 📌 About
**Blessing Generator** is a lightweight, high-performance web tool built to deliver random positive affirmations and blessings. Featuring a sophisticated deep-purple color palette and glass-morphism UI, it provides a serene and aesthetic experience for users looking for daily inspiration.

## ✨ Key Features
- **Immersive Design**: A carefully curated deep-purple aesthetic that reduces eye strain.
- **Glass-Morphism UI**: Modern, sleek, and responsive interface design.
- **Zero Dependencies**: Pure vanilla JavaScript for maximum performance and minimal bloat.
- **Randomized Logic**: Sophisticated selection algorithm ensuring a unique experience every time.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3 (Modern Flexbox & Backdrop Filter)
- **Scripting**: Vanilla JavaScript (ES6+)
- **Build**: Pure static deployment
